# personal-firewall
A lightweight personal firewall that monitors and filters incoming and outgoing network traffic based on user-defined rules.
## Features
- Live packet sniffing
- Protocol analysis (TCP, UDP, ICMP)
- Rule-based filtering
- Traffic logging
- Linux iptables integration

## Tech Stack
- Python 3
- Scapy
- iptables (Linux)

## Skills Demonstrated
- Network Security
- Packet Filtering
- Protocol Analysis
- System Programming

## Run
sudo python3 firewall.py
